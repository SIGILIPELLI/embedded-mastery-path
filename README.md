# Embedded Systems Mastery Path

A free, structured, module-wise embedded systems training program — entry
level to master level, starting with Arduino/ESP32 and C/C++, with real
compilable code in every module and a hands-on project at the end of each
level. Every Level 1 example runs in the free [Wokwi online
simulator](https://wokwi.com/), so no hardware is required to start.

**Live site:** https://sigilipelli.github.io/embedded-mastery-path/

## Contents

- **Level 1 · Entry** — Arduino IDE & Wokwi setup, C/C++ for microcontrollers, digital I/O, analog & PWM, serial/UART, I2C/SPI sensors, timers & interrupts, memory & power, ESP32 WiFi, capstone (environment monitor)
- **Level 2 · Intermediate** — FreeRTOS tasks & queues, ESP-IDF, BLE, MQTT, OTA updates, advanced peripherals, filesystems, watchdogs, debugging & unit testing
- **Level 3 · Advanced** — bare-metal ARM Cortex-M/STM32, registers & linker scripts, RTOS internals, DMA, low-power design, NVIC, PCB basics (KiCad), protocol deep dives, bootloaders
- **Level 4 · Master** — production firmware architecture, secure boot & OTA at scale, TinyML/Edge AI, RF protocol design, HIL testing & CI, functional safety, manufacturing, fleet management

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
